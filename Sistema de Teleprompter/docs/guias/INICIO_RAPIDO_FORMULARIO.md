# ⚡ Inicio Rápido - Formulario de Pruebas

## 🎯 En 3 Pasos

### 1. Abrir el Formulario
```
📁 Doble clic en: test-form.html
```

### 2. Completar Datos Básicos
- ✏️ Tu nombre
- 📅 Fecha (ya viene llenada)
- 🔢 Versión de la app que estás probando

### 3. Empezar a Probar
- Click en una categoría para expandirla
- Prueba la funcionalidad en el teleprompter
- Marca: ✅ Pass | ❌ Fail | ➖ N/A
- Agrega observaciones si falla

## 📋 3 Botones de Resultado

| Botón | Cuándo Usar | Color |
|-------|-------------|-------|
| ✅ **Pass** | Funciona correctamente | Verde |
| ❌ **Fail** | No funciona / tiene bug | Rojo |
| ➖ **N/A** | No aplica / no probado | Gris |

## 💾 Auto-Guardado

- ⏰ Se guarda automáticamente cada 30 segundos
- 💡 También al marcar cada prueba
- 🔒 Datos seguros en tu navegador

## 📄 Exportar Reporte

```
1. Click en "📄 Exportar Resultados"
2. Se descarga: Reporte_Pruebas_TuNombre_2025-10-01.html
3. Abre el archivo para ver el reporte completo
4. Envía por email al equipo
```

## 🚨 Si Encuentras un Bug

Cuando marques ❌ **Fail**, SIEMPRE agrega observaciones:

```
❌ Ejemplo de Observación Completa:

"El botón Play no inicia el desplazamiento.
 
Pasos:
1. Cargar un script
2. Click en Play
3. El texto no se mueve

Navegador: Chrome 118
Reproducible: Siempre (100%)"
```

## ⚡ Atajos Útiles

- **Ctrl+P**: Imprimir
- **F5**: Recargar (no pierde datos guardados)
- **ESC**: Cerrar modales
- **Tab**: Navegar entre campos

## 🆘 Problemas Comunes

| Problema | Solución |
|----------|----------|
| No puedo guardar | Ingresa tu nombre primero |
| No veo mis datos | Revisa que sea el mismo navegador |
