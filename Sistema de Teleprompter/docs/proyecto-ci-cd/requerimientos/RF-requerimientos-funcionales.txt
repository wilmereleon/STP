REQUERIMIENTOS FUNCIONALES - SISTEMA DE TELEPROMPTER
======================================================

FECHA: Enero 2025
VERSIÓN: 2.0 (Con Backend y Docker)

═══════════════════════════════════════════════════════
RF-001: MÓDULO DE AUTENTICACIÓN Y USUARIOS
═══════════════════════════════════════════════════════

RF-001.1: El sistema debe permitir registro de usuarios con rol "Producer"
RF-001.2: El sistema debe validar credenciales mediante JWT
RF-001.3: Los tokens JWT deben expirar después de 24 horas
RF-001.4: El sistema debe permitir acceso sin autenticación para rol "Operator"
RF-001.5: El sistema debe mantener sesión activa usando localStorage/cookies
RF-001.6: El sistema debe permitir logout manual
RF-001.7: Las contraseñas deben hashearse con bcryptjs (10 salt rounds)

═══════════════════════════════════════════════════════
RF-002: MÓDULO DE SCRIPTS (CRUD)
═══════════════════════════════════════════════════════

RF-002.1: El sistema debe permitir crear nuevos scripts con título y contenido
RF-002.2: El sistema debe permitir editar scripts existentes
RF-002.3: El sistema debe permitir eliminar scripts (soft delete recomendado)
RF-002.4: El sistema debe listar todos los scripts con paginación (20 por página)
RF-002.5: El sistema debe buscar scripts por título, contenido o tags (full-text search)
RF-002.6: El sistema debe filtrar scripts por categoría, prioridad y estado
RF-002.7: El sistema debe asignar metadatos: categoría, prioridad, duración, tags
RF-002.8: El sistema debe calcular duración estimada automáticamente (180 palabras/minuto)
RF-002.9: El sistema debe guardar historial de versiones (últimas 10 versiones)
RF-002.10: El sistema debe permitir restaurar versiones anteriores
RF-002.11: El sistema debe marcar autor de creación y última modificación
RF-002.12: El sistema debe permitir agregar notas del productor (no visibles en teleprompter)

RF-006: MÓDULO DE MACROS Y SHORTCUTS
═══════════════════════════════════════════════════════

RF-006.1: El sistema debe asignar teclas F1-F12 a funciones configurables
RF-006.2: El sistema debe permitir macros de navegación (PageUp/PageDown)
RF-006.3: El sistema debe permitir configurar macros desde panel de configuración

RF-006.5: El sistema debe ejecutar macros con respuesta inmediata (<100ms)
RF-006.6: El sistema debe prevenir conflictos de shortcuts del sistema

Macros predefinidos:
- F11: Play/Pause
- F12: Reset
- PageUp: Script anterior
- PageDown: Script siguiente
- Ctrl+F: Buscar en script actual
- Ctrl++: Aumentar fuente
- Ctrl+-: Disminuir fuente
