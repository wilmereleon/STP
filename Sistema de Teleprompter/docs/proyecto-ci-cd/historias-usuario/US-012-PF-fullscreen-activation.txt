# US-012: Activación de Modo Pantalla Completa | Epic: Pantalla Completa | 🟡 ALTA | 3 pts | Sprint 1

## 📋 Descripción

**Como** operador del teleprompter  
**Quiero** activar el modo pantalla completa con un botón o tecla F11  
**Para** maximizar el área de lectura y eliminar distracciones

**Origen:** Bug PF-1 del Reporte de Pruebas (23 Oct 2025)  
**Problema:** Botón de fullscreen no funciona o tiene comportamiento inconsistente

---

- [ ] Botón "Fullscreen" visible y funcional en TeleprompterControls
- [ ] Tecla F11 activa/desactiva fullscreen
- [ ] Transición suave a fullscreen (sin parpadeos)
- [ ] Oculta barras de navegación y UI del browser

## 📝 Notas

### Fullscreen API Implementation

```typescript
const useFullscreen = () => {
  const [isFullscreen, setIsFullscreen] = useState(false);
  
  const enterFullscreen = async () => {
    const element = document.documentElement;
    
    try {
      if (element.requestFullscreen) {
        await element.requestFullscreen();
      } else if (element.webkitRequestFullscreen) { // Safari
        await element.webkitRequestFullscreen();
      } else if (element.mozRequestFullScreen) { // Firefox
        await element.mozRequestFullScreen();
      } else if (element.msRequestFullscreen) { // IE11
        await element.msRequestFullscreen();
      } else {
        throw new Error('Fullscreen API no soportada');
      }
      
      setIsFullscreen(true);
    } catch (error) {
      console.error('Error al entrar en fullscreen:', error);
      // Mostrar mensaje al usuario
      showNotification('Tu navegador no soporta modo fullscreen', 'warning');
    }
  };
  
  const exitFullscreen = async () => {
    try {
      if (document.exitFullscreen) {
        await document.exitFullscreen();
      } else if (document.webkitExitFullscreen) {
        await document.webkitExitFullscreen();
      } else if (document.mozCancelFullScreen) {
        await document.mozCancelFullScreen();
      } else if (document.msExitFullscreen) {
        await document.msExitFullscreen();
      }
      
      setIsFullscreen(false);
    } catch (error) {
      console.error('Error al salir de fullscreen:', error);
    }
  };
  
  const toggleFullscreen = () => {
    if (isFullscreen) {
      exitFullscreen();
    } else {
      enterFullscreen();
    }
  };
  
  // Listener para detectar cambios (ej: usuario presiona ESC)
  useEffect(() => {
    const handleFullscreenChange = () => {
      const isNowFullscreen = !!(
        document.fullscreenElement ||
        document.webkitFullscreenElement ||
        document.mozFullScreenElement ||
        document.msFullscreenElement
      );
      
      setIsFullscreen(isNowFullscreen);
    };
    
    document.addEventListener('fullscreenchange', handleFullscreenChange);
    document.addEventListener('webkitfullscreenchange', handleFullscreenChange);
    document.addEventListener('mozfullscreenchange', handleFullscreenChange);
    document.addEventListener('MSFullscreenChange', handleFullscreenChange);
    
    return () => {
      document.removeEventListener('fullscreenchange', handleFullscreenChange);
      document.removeEventListener('webkitfullscreenchange', handleFullscreenChange);
      document.removeEventListener('mozfullscreenchange', handleFullscreenChange);
      document.removeEventListener('MSFullscreenChange', handleFullscreenChange);
    };
  }, []);
  
  return { isFullscreen, enterFullscreen, exitFullscreen, toggleFullscreen };
};
```

### Keyboard Shortcut (F11)

```typescript
useEffect(() => {
  const handleKeyDown = (e: KeyboardEvent) => {
    if (e.key === 'F11') {
      e.preventDefault(); // Evitar fullscreen del browser
      toggleFullscreen();
    }
  };
  
  window.addEventListener('keydown', handleKeyDown);
  
  return () => {
    window.removeEventListener('keydown', handleKeyDown);
  };
}, [toggleFullscreen]);
```

### Cursor Auto-Hide

```typescript
const useCursorAutoHide = (delay = 2000) => {
  const [cursorVisible, setCursorVisible] = useState(true);
  const timeoutRef = useRef<number>();
  
  useEffect(() => {
    const handleMouseMove = () => {
      setCursorVisible(true);
      
      if (timeoutRef.current) {
        clearTimeout(timeoutRef.current);
      }
      
      timeoutRef.current = window.setTimeout(() => {
        setCursorVisible(false);
      }, delay);
    };
    
    document.addEventListener('mousemove', handleMouseMove);
    
    return () => {
      document.removeEventListener('mousemove', handleMouseMove);
      if (timeoutRef.current) {
        clearTimeout(timeoutRef.current);
      }
    };
  }, [delay]);
  
  return cursorVisible;
};

// En el componente:
const cursorVisible = useCursorAutoHide(2000);

<div className={`teleprompter ${!cursorVisible ? 'cursor-hidden' : ''}`}>
  {/* content */}
</div>
```
