## 🧪 Casos de Prueba

### Caso 1: FPS durante scroll automático
- **Dado:** FloatingWindow en Play con scrollSpeed 50
- **Cuando:** Monitoreo FPS con Chrome DevTools
- **Entonces:** FPS constante en 60fps (± 2fps)

### Caso 2: CPU usage normal
- **Dado:** FloatingWindow abierta en idle (sin scroll)
- **Cuando:** Monitoreo CPU usage
- **Entonces:** CPU < 5%

### Caso 3: CPU usage durante scroll
- **Dado:** FloatingWindow en Play
- **Cuando:** Monitoreo CPU usage
- **Entonces:** CPU < 10%

### Caso 4: Memory leaks - Abrir/cerrar repetido
- **Dado:** Heap snapshot inicial
- **Cuando:** Abro y cierro FloatingWindow 20 veces
- **Entonces:** Heap size final ≈ inicial (± 10%)

### Caso 5: Memory usage durante uso prolongado
- **Dado:** FloatingWindow en uso continuo
- **Cuando:** Monitoreo memory después de 4 horas
- **Entonces:** Memory no crece más de 50MB desde inicio

### Caso 6: Cleanup de event listeners
- **Dado:** FloatingWindow con múltiples listeners
- **Cuando:** Cierro la ventana
- **Entonces:** DevTools muestra 0 listeners activos de esa ventana

### Caso 7: Cleanup de timers
- **Dado:** autoScroll activo con setInterval
- **Cuando:** Pause y unmount
- **Entonces:** clearInterval ejecutado, no timers activos

### Caso 8: Renders innecesarios
- **Dado:** React Profiler activo
- **Cuando:** Cambio scrollTop 60 veces/segundo
- **Entonces:** Solo componentes afectados se re-renderizan

### Caso 9: WebSocket cleanup
- **Dado:** WebSocket conectado con 5 listeners
- **Cuando:** Cierro FloatingWindow
- **Entonces:** socket.disconnect() llamado, listeners removidos

## 📝 Notas

### Memory Leak Checklist

```typescript
// ❌ MAL - Memory leak
useEffect(() => {
  const interval = setInterval(() => {
    updateScrollPosition();
  }, 16);
  // FALTA: return cleanup
}, []);

// ✅ BIEN - Sin memory leak
useEffect(() => {
  const interval = setInterval(() => {
    updateScrollPosition();
  }, 16);
  
  return () => {
    clearInterval(interval); // Cleanup
  };
}, []);
```

### Event Listeners Cleanup

```typescript
// ❌ MAL - Listener no removido
useEffect(() => {
  window.addEventListener('resize', handleResize);
}, []);

// ✅ BIEN - Listener removido
useEffect(() => {
  window.addEventListener('resize', handleResize);
  
  return () => {
    window.removeEventListener('resize', handleResize);
  };
}, [handleResize]);
```

### RequestAnimationFrame Cleanup

```typescript
// ❌ MAL - RAF no cancelado
useEffect(() => {
  const animate = () => {
    updateScroll();
    requestAnimationFrame(animate);
  };
  requestAnimationFrame(animate);
}, []);

// ✅ BIEN - RAF cancelado
useEffect(() => {
  let rafId: number;
  
  const animate = () => {
    updateScroll();
    rafId = requestAnimationFrame(animate);
  };
  
  rafId = requestAnimationFrame(animate);
  
  return () => {
    cancelAnimationFrame(rafId);
  };
}, []);
```

### WebSocket Cleanup

```typescript
// ❌ MAL - Socket no desconectado
useEffect(() => {
  socket.on('state:update', handleUpdate);
  socket.connect();
}, []);

// ✅ BIEN - Socket limpio
useEffect(() => {
  socket.on('state:update', handleUpdate);
  socket.connect();
  
  return () => {
    socket.off('state:update', handleUpdate);
    socket.disconnect();
  };
}, [handleUpdate]);
```

### React Optimization

```typescript
// Evitar re-renders innecesarios
const TeleprompterContent = React.memo(({ content, fontSize }) => {

  // Solo re-renderizar si cambió content o fontSize
  return prevProps.content === nextProps.content &&

```

### Performance Testing Script

```javascript
// Ejecutar en Chrome DevTools Console
async function testMemoryLeaks() {
  console.log('🧪 Test de Memory Leaks');
  
  const initialHeap = performance.memory.usedJSHeapSize;
  console.log(`Heap inicial: ${(initialHeap / 1024 / 1024).toFixed(2)} MB`);
  
  // Abrir/cerrar ventana 20 veces
  for (let i = 0; i < 20; i++) {
    await openFloatingWindow();
    await sleep(1000);
    await closeFloatingWindow();
    await sleep(1000);
    
    if (i % 5 === 0) {
      const currentHeap = performance.memory.usedJSHeapSize;
      console.log(`Iteración ${i}: ${(currentHeap / 1024 / 1024).toFixed(2)} MB`);
    }
  }
  
  // Forzar GC (requiere flag --expose-gc)
  if (global.gc) {
    global.gc();
    await sleep(2000);
  }
  
  const finalHeap = performance.memory.usedJSHeapSize;
  const diff = finalHeap - initialHeap;
  console.log(`Heap final: ${(finalHeap / 1024 / 1024).toFixed(2)} MB`);
  console.log(`Diferencia: ${(diff / 1024 / 1024).toFixed(2)} MB`);
  
  if (diff > 50 * 1024 * 1024) { // 50MB
    console.error('❌ Memory leak detectado!');
  } else {
    console.log('✅ Sin memory leaks');
  }
}
```
